== Description ==

Add to Sharedaddy support service.

* [Twitter](https://twitter.com/about/resources/tweetbutton)（日本語化）
* [Google +1](http://www.google.com/intl/ja/webmasters/+1/button/)
* [Google Buzz](http://www.google.com/buzz/api/admin/configPostWidget)
* [はてなブックマーク](http://b.hatena.ne.jp/guide/bbutton)
* [mixiチェック](http://developer.mixi.co.jp/connect/mixi_plugin/mixi_check/)
* [Evernote](http://www.evernote.com/about/intl/jp/developer/sitememory/)

**PHP5 REQUIRE**

== Installation ==

Install [Sharedaddy](http://wordpress.org/extend/plugins/sharedaddy/) or [Jetpack](http://wordpress.org/extend/plugins/jetpack/) plugin.

Upload the this plugin to your blog, Activate it.
